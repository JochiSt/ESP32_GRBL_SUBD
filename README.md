# ESP32_GRBL_SUBD

ESP32 GRBL adapter to SUB-D25

adapter to use the [ESP32 GRBL port](https://github.com/bdring/Grbl_Esp32) with
usual CNC machines, which are controlled via LPT (SUB-D25)
